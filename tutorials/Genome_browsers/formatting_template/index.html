    Questions and Answers

    It is useful to number questions with the section number they appear in so participants can easily refer to the number when asking a question. Indentation is important to display correctly.

    !!! question "Question 1.1" What is the full path name of your home directory?

    ??? hint "Hint"
    Remember your Current Working Directory starts in your home directory.

??? success "Answer"
    You can find out the full path name of the current working directory with the `pwd` command. Your home directory will look something like this:

    ```Bash
    $ pwd
    /home/trainingXX

    ```
